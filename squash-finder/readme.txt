Squash Finder is a WordPress plugin designed to help users find squash courts near their location.

== Description ==

Squash Finder is a WordPress plugin designed to help users find squash courts near their location. Whether you're a seasoned player or just starting out, Squash Finder makes it easy to locate courts so you can enjoy a game of squash whenever you want.

Features include:
- Search functionality based on location
- Interactive map display
- Detailed court information including address, contact details, and availability
- User reviews and ratings for courts
- Customizable filters for search results

== Installation ==

1. Download the Squash Finder plugin zip file from the official WordPress plugin repository.
2. Log in to your WordPress admin panel.
3. Navigate to Plugins > Add New.
4. Click on the "Upload Plugin" button and select the plugin zip file you downloaded.
5. Click on the "Install Now" button and then activate the plugin.

== Usage ==

To integrate Squash Finder into your WordPress site, simply add the [squash_finder] shortcode to any post or page where you want the search functionality to appear. Users can then follow these steps to find squash courts:
1. Visit the page or post where you added the [squash_finder] shortcode.
2. Enter your location or enable location services to automatically detect your current location.
3. Use the search filters to refine your search criteria if needed.
4. Browse the interactive map or view the list of available squash courts.
5. Click on a court to view detailed information, including address, contact details, and user reviews.
6. Select a court and start playing squash!
